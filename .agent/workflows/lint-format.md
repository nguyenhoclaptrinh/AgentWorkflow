# LINT & FORMAT WORKFLOW

## 1. Kiểm tra cấu hình
- Xác định eslint / prettier / flake8 / black...

## 2. Chạy lint (safe)
- Chạy lint ở chế độ check, không auto-fix trước

## 3. Format code
- Auto-format toàn bộ file liên quan

## 4. Verify
- Đảm bảo không còn warning nghiêm trọng

## 5. Commit
- Commit message:
  chore: lint & format code
